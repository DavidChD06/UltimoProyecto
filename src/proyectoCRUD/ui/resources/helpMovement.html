Account Movements: Help

General Description

This window allows you to view and manage the financial movements of a specific account. Users can track their transaction history in the central table, register new operations, or revert the last action performed. Additionally, the updated balance is displayed at the top.

Operations

New movement

To register a new operation, the user must select the Type of movement (Deposit or Payment) from the dropdown menu and enter the value in the Amount field. Once the data is entered, click the New movement button. If the operation is successful, the movement will appear in the table and the balance will be updated automatically.

Undo last movement

If the user needs to revert the last action performed, click the Undo last movement button. The system will remove the most recent record from the table and restore the account's previous balance.

Data Visualization

The central table displays the list of movements with the following details:

  • DATE: The date the movement was recorded.
  • AMOUNT: The value of the operation.
  • TYPE: The category or type of movement(Deposit/Payment).
  • BALANCE: The resulting balance after the operation.
If there are no records, the table would be empty until you create a movemet

Exit

Clicking the Exit button will close the current view and redirect you back to the Accounts window.
