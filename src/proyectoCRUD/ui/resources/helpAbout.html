Bank Management Application

The Bank Management Application is a comprehensive desktop system developed in JavaFX designed to manage personal banking data securely. It follows a Client-Server architecture, synchronizing data in real-time with a RESTful backend.

1. User Access & Security

Security is the core of the application logic:

  • Sign In: Authenticates users via Email and Password against the server database.
  • Sign Up: New users can register with strict validation for personal fields (ZIP code, Phone format, Email pattern).
  • Password Management: Includes a dedicated module to update credentials securely, enforcing complexity rules (uppercase, numbers, length).

2. Account Management

The main dashboard allows complete control over bank accounts:

  • Real-time View: Displays account details including Balance, Type, and Credit Line.
  • Creation: Users can open new accounts instantly. The system auto-generates IDs and validates initial balances.
  • Editing: Allows modification of descriptions and credit lines (for Credit accounts only) directly in the table.
  • Deletion: Logic prevents deleting accounts that still have active movements.

3. Transaction Processing (Movements)

Each account has a detailed history of movements accessible via the interface:

  • Operations: Supports Deposits and Payments.
  • Balance Updates: The account balance is automatically recalculated locally and on the server after every transaction.
  • Undo Function: Users can reverse the last operation performed if needed.

4. Technical Specifications

  • Frontend: JavaFX 8 (FXML & CSS).
  • Backend Communication: Jersey REST Client (XML).
  • Design Pattern: Model-View-Controller (MVC).
  • Features: Contextual Help, Exception Handling, and Input Validation.
